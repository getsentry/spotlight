---
title: ConsoleIntegration
description: Interface of ConsoleIntegration
---

The console integration allows you to easily render console logs within Spotlight.

:::note[Warning - Experimental]

This integration is currently experimental and may change or be abandoned in the future.
It still lacks a lot of features. Feel free to [contribute](/docs/contribute)!

:::

```ts ins=/console[\(]?[\)]?/
import { init, console } from "@spotlightjs/spotlight";

init({
  integrations: [console()]
});
```
