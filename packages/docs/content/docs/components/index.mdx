# Components

Explore all the components available in Spotlight UI. Each component is designed for building observability and distributed tracing experiences.

<Cards>
  <Card title="SpanTree" href="/docs/components/span-tree">
    Hierarchical waterfall visualization for distributed trace spans. Perfect for displaying the timing and hierarchy of spans within a trace.
  </Card>
  <Card title="TraceItem" href="/docs/components/trace-item">
    Summary row component for displaying distributed trace information. Ideal for trace lists and overviews.
  </Card>
</Cards>

## Installation

All components can be installed using the shadcn CLI with our custom registry:

```bash
pnpm dlx shadcn@latest add <component-name> -r https://spotlightjs.com/registry.json
```

Or you can manually copy the component files. See each component's documentation for specific installation instructions.
